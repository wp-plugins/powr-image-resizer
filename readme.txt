== Installation ==
To install POWr Image Resizer
<br><br>
1. Upload 'powr-image-resizer.php' to the '/wp-content/plugins/' directory
<br>
2. Activate the plugin through the 'Plugins' menu in WordPress
<br><br>
To add POWr Image Resizer as a widget, go to **Appearance->Widgets**, and drag POWr Image Resizer to where you'd like it to appear.
<br><br>
To add Image Resizer within a post or a page, simply write the shortcode **[powr-image-resizer label='MY LABEL']** (including the brackets) where you'd like your Image Resizer to appear. Then visit your page!
<br><br>
Want a more detailed tutorial? Visit [How to add Image Resizer to your Wordpress site](http://www.powr.io/tutorials/how-to-add-image-resizer-plugin-to-your-wordpress-site)
